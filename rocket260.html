Welcome! Rocket 260 is an app where you can draw rockets before you build the models. This calculates the Center of Gravity (COM) and Center of Pressure (CP) for you, as you add and remove parts.

Save and load files to edit later, and build anywhere, as this app needs no internet connection.

Donations (Even the smallest ones) help fund this project

Tools

Edit Add points. Points are automatically connected to preview the shape
Erase Erase points you've drawn with the draw tool
Build Builds the points you've drawn into a shape with weight
Clear Clears the last built shape
Top Nothing for now
Save/Load Save and load rockets to/from files. All save files are listed on the left side. Just type the file name and hit Enter.
Weight Use num keys and backspace to set the weight of the part you are building before you press build
